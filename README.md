# My Blog Template

I use the template for my personal blog. [(dincer.info)](http://dincer.info)

## Main Features

- Bootstrap v2.3.2
- jQuery v1.9.0
- Glyphicon
- Google Fonts
- Fully Responsive
- Cross Browser Support

## Pages

- Home
- About
- Blog Post
